# Autism

## Description:
Ehtiwa is an intelligent support platform designed to help children and parents through interactive tracking tools, supportive community features, and educational activities.
The platform provides a safe and user-friendly environment where parents can monitor child development, share experiences, and receive personalized support powered by AI technologies.



## Features


- User authentication and secure account management.
- Child profile creation and management.
- AI-powered behavioral observation and analysis.
- Behavioral tracking and progress monitoring.
- Video upload and observation analysis.
- Personalized support plans and activity scheduling.
- Interactive educational and developmental activities.
- Community discussions and parent interaction system.
- Hashtag-based discussion organization and browsing.
- Dedicated discussion pages with comments and engagement.
- Notifications and reminders for activities and updates.
- Search and filtering for discussions and content.
- Responsive design for mobile, tablet, and desktop devices.
- Secure data handling and privacy protection.
- Admin dashboard for managing users, reports, and content.
